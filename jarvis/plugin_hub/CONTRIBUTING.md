# Plugin Hub Contribution Guidelines

Thank you for contributing to the Jarvis Plugin Hub! To submit a plugin:

1. Ensure your plugin includes a manifest describing name, version, author, description, and dependencies.
2. Submit your plugin via the `/api/plugins/submit` endpoint. Your submission will enter a moderation queue.
3. Administrators review pending submissions and approve or reject them.
4. Approved plugins become available for installation by authorized users.

Please keep plugins secure and avoid malicious code. All contributions are subject to review and may be removed if they violate community standards.
