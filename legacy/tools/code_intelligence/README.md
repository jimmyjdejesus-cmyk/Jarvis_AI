# Code Intelligence Module

This folder contains the code intelligence engine, UI, parsers, database, and related utilities for AI-powered code completion and analysis.

## Structure
- `engine.py`: Core engine logic
- `ui.py`: Streamlit UI components
- `database.py`: Database operations
- `parsers.py`: Language-specific parsers
- `config.yaml`: Configuration for models and features
- `tests/`: Unit and integration tests

## Usage
Import and use the engine and UI components as needed in your application.
